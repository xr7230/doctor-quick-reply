# 医患沟通助手 V3.0

基于V2版本全面重构的新版本，带来更优质的用户体验。

## 新功能

### 体验提升
- **深色模式**：支持跟随系统或手动切换，保护医生夜间用眼
- **历史记录**：IndexedDB存储最近20条生成记录，随时追溯
- **模板收藏**：常用输入保存为模板，一键复用
- **一键复制全部**：格式化复制三个版本

### 界面优化
- **Tailwind CSS**：专业级样式，响应式设计
- **手机/平板适配**：移动端布局优化
- **状态指示更明显**：API模式/模拟模式一目了然

### 技术改进
- **TypeScript**：完整类型支持
- **IndexedDB**：本地大数据量存储
- **组件化**：代码结构更清晰

## 快速开始

### 一键启动
双击 `start.bat` 启动应用

### 手动启动
```bash
cd d:\doctor\3
npm install
npm run dev
```

### 使用说明

1. **模拟模式**：无需配置，直接可用
2. **API模式**：点击右上角设置，配置API密钥
3. **生成回复**：输入医生要点，选择场景，生成建议
